ホトカプラ

説明

明るさを読み取ることができるセンサーです

使用方法

digitalRead 関数を使用すると HIGH / LOW で値を読み取ることができます

analogRead 関数を使用するとアナログ値で読み取ることができます

明るさの具合で処理を変更する問題は出題されないので、 digitalRead 関数を使用することが一般的です

サンプルコード

if (digitalRead(ピン番号) == HIGH) {
    // HIGHの時の処理
} else {
    // LOWの時の処理
}
